Mike is a customizable Discord bot with a lot of plugins:
* Logs
* Music
* Economy
* Leveling
* Starboard
* Moderation
* Suggestions
* Chat Filters
* Custom Commands
* Welcome Message
* Image Generation
* Twitch, Reddit Notifications
